Detailed Description

Collapsible, draggable node type entry in the palette.

Each DropBar represents one node type (e.g., "Add", "Branch", "Output"). Users can drag from this entry to spawn a new node of that type in the editor. The associated NodeType is embedded in the drag-and-drop payload for proper node instantiation.
