Detailed Description

Visual node representation for the node graph.

Defines the runtime representation of a node in the visual editor. Nodes contain pins (connection points), editable fields, and state for positioning and lifecycle management.

bool DrawVisualNode ( VisualNode & n)

Render a visual node on the canvas.

Parameters
nThe node to render

Draws the node using imgui-node-editor primitives:

  • Node body with title
  • Input pins on the left
  • Output pins on the right
  • Editable fields in the center

Must be called within an ed::Begin() / ed::End() block.
