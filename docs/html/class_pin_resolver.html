Detailed Description

Fast lookup for input pin → source mappings.

Pre-processes the link list into a hash map for O(1) lookups during recursive AST construction. GraphCompiler queries this to find: "what AST subtree should I build for this input pin?"

Also provides node ID → VisualNode* lookups for convenience.

Usage:

PinResolver resolver;
resolver.Build(nodes, links);
const PinSource* src = resolver.Resolve(inputPinId);
if (src && src->node) {
// Found upstream node feeding this input
const Pin& srcPin = src->node->outPins[src->pinIdx];
}
