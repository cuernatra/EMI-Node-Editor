Detailed Description

Main application class that manages the render window, events, and editor layer.

Orchestrates the application lifecycle: creates and manages the SFML window, processes events, updates the UI state, and renders frames at 60fps. Integrates ImGui-SFML for immediate mode GUI rendering.
