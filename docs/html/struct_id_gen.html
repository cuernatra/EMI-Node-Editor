Detailed Description

Simple ID generator for graph elements.

Generates unique sequential IDs for nodes, pins, and links in the visual graph editor. Uses a single shared counter to ensure all IDs are unique across different element types.

Note
IDs start at 1 and increment monotonically. ID 0 is reserved as invalid.
