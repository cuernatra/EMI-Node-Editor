\doxysubsection{Detailed Description}
Fast lookup for input pin → source mappings. 

Pre-\/processes the link list into a hash map for O(1) lookups during recursive AST construction. \doxylink{class_graph_compiler}{Graph\+Compiler} queries this to find:\+ "{}what AST subtree should I build for this input pin?"{}

Also provides node ID → Visual\+Node\texorpdfstring{$\ast$}{*} lookups for convenience.
