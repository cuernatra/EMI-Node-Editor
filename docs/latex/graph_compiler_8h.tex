Graph-\/to-\/\+AST compilation for EMI-\/\+Script execution. 

Converts visual node graphs to EMI-\/\+Script AST (Abstract Syntax Tree) for compilation and execution via the EMI runtime.

\item \doxylink{class_graph_compiler}{Graph\+Compiler} recursively builds AST nodes from visual nodes
