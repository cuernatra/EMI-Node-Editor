\doxysubsection{Detailed Description}
Graph file I/\+O operations. 

Provides static methods for saving and loading complete graph state to/\+from disk. The file format is a simple text-\/based representation that stores:\+
\begin{DoxyItemize}
\item Node IDs, types, positions, and field values
\item \doxylink{struct_pin}{Pin} IDs and their associations
\item \doxylink{struct_link}{Link} IDs and connections
\item Next available ID for the generator 
\end{DoxyItemize}
