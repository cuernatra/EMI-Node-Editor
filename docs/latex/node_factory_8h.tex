\doxysubsection{Detailed Description}
Node instance creation from registry descriptors. 

Provides factory functions for creating \doxylink{struct_visual_node}{Visual\+Node} instances from their descriptors. Handles both fresh node creation (with new IDs) and deserialization (with existing IDs).
