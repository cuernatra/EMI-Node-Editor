\doxysubsection{Detailed Description}
Identifies the source of a connected input pin. 

Represents where an input pin\textquotesingle{}s value comes from:\+
\begin{DoxyItemize}
\item node:\+ The upstream \doxylink{struct_visual_node}{Visual\+Node} with the output pin
\item pin\+Idx:\+ Index into node-\/\texorpdfstring{$>$}{>}out\+Pins (which output pin on that node)
\end{DoxyItemize}

Used during recursive AST building to traverse from outputs back through inputs, accumulating the expression tree.
