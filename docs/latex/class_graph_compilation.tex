\doxysubsection{Detailed Description}
High-\/level graph compilation and execution interface. 

Manages the EMI-\/\+Script VM environment and provides a simple compilation interface for the editor. Validates the graph, compiles to AST, and prepares for EMI\textquotesingle{}s internal ASTWalker compilation.

Currently creates and prints the AST. Future integration will use EMI\textquotesingle{}s Parser::\+\+Parse\+AST() and internal execution APIs.
