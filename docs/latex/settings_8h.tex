\doxysubsection{Detailed Description}
Persistent application settings. 

Loads and saves settings to settings.\+json in the working directory. Call \doxylink{namespace_settings_aca28e31f4a12cc9f0c2e522818784a5d}{Settings::\+\+Load()} on startup and \doxylink{namespace_settings_a86e6fc9057d40adb5ce297b28f797a68}{Settings::\+\+Save()} on shutdown.

Adding a new setting:\+
\begin{DoxyEnumerate}
\item Add an inline variable with a sensible default value
\item Add a line in Load() using s.\+value("{}key"{}, default\+Value)
\item Add a line in Save() using s\+["{}key"{}]\+ = member 
\end{DoxyEnumerate}
