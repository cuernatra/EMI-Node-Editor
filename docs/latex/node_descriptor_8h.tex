\doxysubsection{Detailed Description}
Static node type definitions and descriptors. 

Defines the structure of node types via descriptors that specify:\+
\begin{DoxyItemize}
\item \doxylink{struct_pin}{Pin} configuration (names, types, directions)
\item Editable fields (names, types, defaults)
\item Display information (labels)
\end{DoxyItemize}

These descriptors are templates used by the factory to create node instances.
