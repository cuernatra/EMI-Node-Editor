\doxysubsection{Detailed Description}
Renders a small preview of a node type. 

Draws a miniature representation of a node showing:\+
\begin{DoxyItemize}
\item Node body with title
\item Input pins on the left (tiny circles)
\item Output pins on the right (tiny circles)
\item Field labels if space allows
\end{DoxyItemize}

Used in the left panel palette to show what each draggable node type looks like before user drags it to canvas.
