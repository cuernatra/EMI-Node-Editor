\doxysubsection{Detailed Description}
Compiles visual graphs to EMI-\/\+Script AST. 

Translates connected node graphs into an Abstract Syntax Tree suitable for EMI-\/\+Script compilation. Each visual node is converted to one or more AST nodes based on its type (operators become expressions, prints become function calls, etc.).

The generated AST is wrapped in a function named k\+Graph\+Function\+Name so that future EMI integration can execute the graph and recover its return value.
