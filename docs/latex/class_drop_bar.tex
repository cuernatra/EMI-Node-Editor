\doxysubsection{Detailed Description}
Collapsible, draggable node type entry in the palette. 

Each \doxylink{class_drop_bar}{Drop\+Bar} represents one node type (e.\+g., "{}\+Add"{}, "{}\+Branch"{}, "{}\+Output"{}). Users can drag from this entry to spawn a new node of that type in the editor. The associated \doxylink{types_8h_acac9cbaeea226ed297804c012dc12b16}{Node\+Type} is embedded in the drag-\/and-\/drop payload for proper node instantiation.
