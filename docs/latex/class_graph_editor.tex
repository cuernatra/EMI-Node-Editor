\doxysubsection{Detailed Description}
Node editor canvas renderer and interaction manager. 

Handles all visual and interactive aspects of the node graph:\+
\begin{DoxyItemize}
\item Drawing nodes with pins and fields
\item Rendering links between pins
\item Processing user interactions (drag, connect, delete)
\item Displaying context menus for spawning/\+removing elements
\item Validating link creation (type compatibility, cycle detection)
\end{DoxyItemize}

Operates on a \doxylink{class_graph_state}{Graph\+State} reference but doesn\textquotesingle{}t own it.
