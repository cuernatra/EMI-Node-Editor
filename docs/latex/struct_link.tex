\doxysubsection{Detailed Description}
Represents a connection (edge) between two pins in the node graph. 

A \doxylink{struct_link}{Link} connects an output pin to an input pin, with a specific data type that determines the visual appearance of the connection wire. Links can be marked as inactive (alive=false) for deletion.
